```
State Transitions:

1. CREATED ? PENDING
   - Invitation created by faculty
   - Both students have "Pending" response

2. PENDING ? ACCEPTED
   - Both students respond "Accepted"
   - Status changes to "Accepted"

3. PENDING ? REJECTED
   - At least one student responds "Rejected"
   - Status changes to "Rejected"

4. ACCEPTED ? COMPLETED
   - Meeting date/time has passed
   - Meeting took place successfully

5. ANY ? CANCELLED
   - Faculty cancels the meeting
   - Can happen at any time before meeting
```
